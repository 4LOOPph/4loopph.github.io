Install

Get the Source File

You can use bower to install or download the zip file here, and locate the .js file in the dist folder:

Native JS
bower install frontier

OR

npm i frontier (NPM Coming Soon..)

OR via CDN
<script src="http://fourloopph.bitbucket.org/developers/bin/analytics/v1/js/frontier.min.js"></script>

OR
<script src="bower_components/frontier/dist/frontier.min.js"></script>
Angular JS
bower install ng-frontier

OR

NPM i ng-frontier (Bower Coming Soon..)

OR via CDN
<script src="http://fourloopph.bitbucket.org/developers/bin/analytics/v1/js/frontier.min.js"></script>
<script src="http://fourloopph.bitbucket.org/developers/bin/analytics/v1/angular/ng-frontier.min.js"></script>

OR
<script src="bower_components/frontier/dist/frontier.min.js"></script>
<script src="bower_components/ng-frontier/dist/ng-frontier.min.js"></script>

Include ng-frontier.js or ng-frontier.min.js in your index.html file after frontier.js and after your AngularJS file (since ngFrontier depends on AngularJS).

Inject as an Angular dependency

Then, include ngFrontier as a dependency in your angular module:

angular.module('sampleApp', ['ngFrontier'])
