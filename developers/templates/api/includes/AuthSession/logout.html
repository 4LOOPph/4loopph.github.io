Logs the current user out of Symphony, destroying the existing session, if any.

Responses
  • Status 200
    Returned if the user was successfully logged out. 

    {
  "response": {
    "msg": "User successfully logout",
    "success": true,
    "result": null
  },
  "statusCode": 200
}
  • Status 401
    Returned if the user was successfully authenticated.
